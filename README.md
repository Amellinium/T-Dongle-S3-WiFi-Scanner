# T-Dongle S3 WiFi Scanner

WiFi scanner for **LilyGO T-Dongle S3** with built-in TFT screen

## Features
- Scan nearby WiFi networks every 5 seconds.
- Show network list and RSSI on TFT display.
- Built-in LED pulses red while scanning.
- Button toggles screen rotation (upright â†” upside-down).

## Requirements
- LilyGO T-Dongle S3 with screen
- Arduino IDE or PlatformIO
- Libraries:
  - WiFi
  - TFT_eSPI
  - FastLED
